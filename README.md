# Flayla Estética 🌿

Landing page da **Flayla Estética** — estética, beleza, cosméticos e cuidados pessoais com produtos naturais.

## 🔗 Site no ar

**https://flayla-estetica.vercel.app**

## ✨ Sobre

Landing page responsiva com seções de Hero, diferenciais, serviços, sobre, produtos, depoimentos e contato. Paleta inspirada na natureza — branco, verde e bege.

### Serviços apresentados
Massagem relaxante · Massagem modeladora · Drenagem linfática · Limpeza de pele profunda · Skincare personalizado · Depilação · Design de sobrancelhas · Spa das mãos & pés

## 🛠️ Tecnologias

- [Next.js 15](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- Fontes: Cormorant Garamond + Jost

## 🚀 Rodando localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## 📦 Build de produção

```bash
npm run build
npm start
```

## ☁️ Deploy

Hospedado na [Vercel](https://vercel.com/), com deploy automático a cada push na branch `main`.
